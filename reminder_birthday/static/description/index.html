Birthday Reminder Notification
  • Configuration for send birthday notification
  • Add date of birth for employee.
  • Greetings can be sent by the automatic cron scheduler and send email and notification odoobot.

Multi-Languages

Have added predefined some languages (English, Spanish/Español) in module, If you want to add other language you can contact me. Use google translator tools for multi-languages so possible translations can be not accurate. I can accept suggest to make more accurate translations.
